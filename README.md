### 重要概念

* Namespace 和 label 管理应用
* Service 暴露服务
* Deployment 部署无状态应用
* StatefulSet 部署有状态中间件
* DaemonSet 部署日志收集器
* ConfigMap 存储配置文件
* Secrets 存储敏感数据
* pv/pvc/storageclass 存储数据
* downwardApi 获取 pod 元数据
* livenessProbe 防止应用假死
* readinessProbe 成功注册服务
* resources 限制资源

### 应用层

* tomcat + war
* springboot

### 中间件

* efk
* skywalking
* prometheus
    
* mysql
